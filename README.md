## 📝 Reflexión
En este proyecto aprendí a estructurar una página web utilizando HTML y a darle estilo con CSS para lograr una interfaz como la solicitaba.

Una de las dificultades que tuve fue centrar correctamente los elementos en la pantalla y lograr que el formulario se viera ordenado. También tuve algunos problemas con la tipografía y la ruta de los archivos, pero lo solucioné revisando la estructura de carpetas y los enlaces del CSS. Para facilitarme el trabajo, descargué la carpeta de la fuente.

Otro de los problemas con los que me encontré fue que algunas clases no se aplicaban correctamente; al revisar el archivo HTML y el CSS descubrí que se debía a errores de escritura, por lo que tuve que prestar más atención a los nombres de las clases y su correcta escritura.

Además, reforcé el uso de propiedades como flexbox, box-shadow y border-radius.

En general, este ejercicio me ayudó a entender mejor cómo crear interfaces funcionales y ordenadas para páginas web.
